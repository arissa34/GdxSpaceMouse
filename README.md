# GdxSpaceMouse

Java wrapper for the 3DConnexion SpaceMouse

![ScreenShot](https://github.com/arissa34/GdxSpaceMouse/blob/master/screens/spacemouse.png?raw=true)

JniGen is used for the C++ code parser & library builder

## Requirements

3DConnexion drive
